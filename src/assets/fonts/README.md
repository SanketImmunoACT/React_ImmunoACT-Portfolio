# Fonts Folder

Place font files here:

## Primary Brand Font:
- `FuturaMdBT-Regular.woff2` - Futura Md BT Regular (preferred format)
- `FuturaMdBT-Regular.woff` - Fallback format
- `FuturaMdBT-Regular.ttf` - TTF format if needed

## Font Weights (if available):
- `FuturaMdBT-Bold.woff2` - Bold weight
- `FuturaMdBT-Medium.woff2` - Medium weight
- `FuturaMdBT-Light.woff2` - Light weight

## Usage:
The font will be loaded via CSS @font-face and set as the primary font family in Tailwind config.
