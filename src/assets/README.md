# Assets Folder Structure

This folder contains all the media assets for the ImmunoACT website organized by component and page.

## Folder Organization:
- `logos/` - Brand logos and company logos
- `images/` - General images organized by sections
- `icons/` - SVG icons and small graphics
- `backgrounds/` - Background images and patterns
- `documents/` - PDFs, brochures, and downloadable content

## File Naming Convention:
- Use kebab-case: `hero-background.jpg`
- Include descriptive names: `patient-kuldeep-testimonial.jpg`
- Add size suffix if multiple sizes: `logo-large.svg`, `logo-small.svg`
